## 1. 设计原则

**核心风格**：当代水墨生机主义（Contemporary Ink Vitalism）

- **留白为王**：大量负空间，体现东方水墨精神
- **材质真实感**：轻微纸张纤维纹理、墨迹渗透感
- **动态自然**：所有动效缓慢、有机、仿物理（避免卡通）
- **极简控制**：默认隐藏多数 UI，专注作品本身
- **一致性**：网页端与桌面端视觉 100% 统一
- **沉浸优先**：画廊/直播模式下 UI 可完全隐形

**关键词**：静谧、呼吸、层积、共生、时光

## 2. 整体布局

### 2.1 默认主界面（Desktop / 大屏）

```
[ 极窄左侧边缘控制条（可隐藏） ]
[                               ]
[          主画布 (Canvas)       ]  ← 占满剩余空间
[                               ]
[ 右侧悬浮控制面板（默认收起）   ]
```

- **主画布**：始终居中，保持 16:9 或 4:3 比例，周围留白
- **左侧边缘条**：垂直极简图标栏（天气、音频、设置）
- **右侧悬浮面板**：默认收起，点击图标展开，可拖拽

### 2.2 全屏展示模式（画廊/直播推荐）
- 所有 UI 隐藏
- 仅右下角极淡水印（城市 + 时间，可关闭）
- Esc 或点击边缘 3 秒退出全屏

### 2.3 移动端布局
- 底部 Tab Bar（画布 / 控制 / 导出）
- 侧边栏改为底部抽屉式
- 优先竖排书法展示

## 3. 关键界面描述

### 3.1 控制面板（右侧悬浮）

**面板分为 4 个 Tab**：

1. **天气 Weather**
   - 当前城市 + 实时四参数大数字显示
   - 手动滑块（温度、湿度、风速、气压）
   - 一键虚拟城市快捷按钮（12 个缩略卡片）
   - “刷新”按钮 + 降级状态徽章（离线/手动）

2. **声音 Audio**
   - 实时频谱可视化（圆形或横向水墨风格）
   - 输入源切换（麦克风 / 文件 / 系统音频）
   - 音量指示环 + BPM 显示（若可用）
   - 频段能量柱状图（低/中/高频）

3. **菌落 Colony**
   - 配色主题切换（3-5 种预设）
   - 粒子密度滑块
   - 生长速度倍率（0.5x ~ 3x）
   - “重生全部菌落”按钮

4. **时间 Time**
   - 氧化速度控制（1x / 10x / 100x）
   - 层积间隔设置
   - 当前作品“年龄”显示
   - “历史回放”时间轴

**交互细节**：
- 面板支持半透明毛玻璃效果（backdrop-filter: blur）
- 拖拽标题栏可移动位置
- 点击画布任意位置收起面板

### 3.2 顶部/底部控制条（可选，常驻极简）

- 左上：Logo + 当前作品标题（可编辑）
- 右上：全屏 / 导出 / 设置
- 底部：当前天气简况 + 音频状态

## 4. 核心交互规范

| 操作 | 交互方式 | 反馈效果 |
|------|----------|----------|
| 切换天气参数 | 滑块拖动 | 书法实时重绘（平滑过渡 800ms） |
| 上传音频文件 | Drag & Drop | 墨迹边缘轻微“震动”反馈 |
| 点击画布 | 单机收起 UI，双击暂停/继续时间线 | 轻微墨晕扩散动画 |
| 长按画布某处 | 弹出该区域历史沉积记录 | 时光倒流层叠预览 |
| 导出 | 一键快照 / 延时摄影 | 墨迹“定格” + 纸张卷起动画 |
| 系统音频捕获 | 桌面端专属开关 | 频谱突然活跃 + 菌落受惊效果 |

**动效原则**：
- 所有过渡默认 400-800ms，easing: `cubic-bezier(0.23, 1, 0.32, 1)`（水墨感）
- 菌落生长使用缓动而非线性
- 氧化过程每 30 分钟使用 HSL 缓慢色相偏移

## 5. 组件设计规范

### 5.1 滑块控件
- 自定义水墨风格滑块（轨道为淡墨线，Thumb 为墨滴形状）
- 数值显示采用毛笔手写数字字体

### 5.2 按钮
- 主要按钮：墨迹描边 + 悬停时轻微晕染
- 次要按钮：纯文字，hover 下划线

### 5.3 频谱可视化
- 风格：水墨粒子柱状或圆形雷达图
- 颜色跟随当前菌落主题

### 5.4 状态指示
- 离线模式：右上角浅褐色 “离线” 印章样式徽章
- 性能模式：轻微警告提示，不打断沉浸

---

## 6. 色彩与排版（沿用 PRD 视觉指南）

- **主背景**：Rice Paper `#F8F1E9` 或深墨 `#0F0F0F`
- **文字**：墨黑为主，重要信息使用 Oxidation Brown
- **字体**：
  - 中文标题：汉仪新魏 / 思源宋体 Bold
  - 英文：EB Garamond 或 Inter
  - UI 文字：思源黑体 + Inter

---

## 7. 响应式与适配

- **桌面 ≥ 1080p**：完整右侧面板
- **平板**：面板改为左侧抽屉
- **手机**：底部 Tab + 全屏优先
- **画廊大屏**：强制全屏模式入口

---

## 8. 错误与空状态

- **无音频输入**：画布显示“静默中，菌落进入休眠…” + 微弱呼吸动画
- **天气加载失败**：显示最后缓存数据 + 明显“手动模式”提示
- **首次使用**：轻量 Onboarding 引导（3 步：选城市 → 开麦克风 → 欣赏）
